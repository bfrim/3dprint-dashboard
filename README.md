# 3D Dash **[WIP]**

A basic web dashboard for 3D printing, built with Flask and Bootstrap.

## How to Run

Install Flask:
```bash
pip install flask
```

Start the app:
```bash
python app.py
```

Visit in your browser:
```bash
http://localhost:5000
```

## Files

- `app.py` — main Flask app  
- `templates/index.html` — homepage  
- `templates/start-print.html` — print page
